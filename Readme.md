# Netflix Login Clone 🎬

A full-stack web application replicating the authentic Netflix login experience. This project demonstrates frontend-to-backend communication, form validation, and mock authentication using a modern JavaScript stack.

## 🚀 Tech Stack

**Frontend:**
* React.js
* Vite (Build Tool)
* Tailwind CSS (Styling & Responsive UI)
* React Router DOM (Navigation)
* Axios (HTTP Client)

**Backend:**
* Node.js
* Express.js
* CORS (Cross-Origin Resource Sharing)

## ✨ Features

* **Authentic UI:** Pixel-perfect replication of the Netflix sign-in page, including the dark UI aesthetics.
* **Mock Authentication:** Backend validation using static, hardcoded credentials without the need for a database.
* **Frontend Validation:** Prevents empty submissions and alerts the user to missing fields.
* **Error Handling:** Displays clear, user-friendly error messages for incorrect credentials directly on the UI.
* **Protected Routing Flow:** Successful login redirects users to a Dashboard, simulating a complete authentication cycle.
